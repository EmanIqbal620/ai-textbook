# Humanoid Robotics Textbook

A comprehensive guide to building intelligent humanoid robots.

## Features

- 📖 6 Complete Modules
- 🤖 AI-Powered Chatbot
- 🎮 Interactive Simulations
- 💻 Hands-on Projects

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run start

# Build for production
npm run build
```

## Modules

1. **ROS 2 Fundamentals** - Robot Operating System
2. **Simulation & Digital Twins** - Gazebo & Unity
3. **AI for Robotics** - NVIDIA Isaac
4. **VLA Systems** - Vision-Language-Action
5. **Hardware Requirements** - CPU, GPU, Sensors
6. **Assessment** - Projects & Exams
